# Archive

Historical phase docs from earlier iterations of edge-reco — kept for context but
no longer load-bearing. The shipped architecture is described in
[`../ARCHITECTURE.md`](../ARCHITECTURE.md); the quickstart in
[`../QUICKSTART.md`](../QUICKSTART.md); the deployment shapes in
[`../DEPLOY.md`](../DEPLOY.md).

What's here:

- `PRD.md`, `TECH_SPEC.md`, `MVP_ROADMAP.md`, `PRE_PIVOT_README.md` — earlier
  product/tech narratives. They predate the local-first / signed-bundle pivot
  and the @edgeproc/browser extraction; treat as history.
- `plans/`, `specs/` — phase 0 + phase 1 plans/specs (proof-of-loop and the
  Wasm scorer experiments). Superseded by the shipped Python + @edgeproc/browser
  implementation.

If you need to see how something used to be, `git log --follow <file>` and the
commit history are the canonical record. This folder is a courtesy.
